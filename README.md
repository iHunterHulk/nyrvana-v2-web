# Nyrvana V2 Web

Personal Infrastructure Dashboard - Next.js 16 implementation with Tailwind v4, Geist fonts, and shadcn/ui components.

This web interface provides a unified dashboard for managing and monitoring your personal infrastructure services including AdGuard, Ntfy, Ollama, Memos, and more.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Type checking:
```bash
npm run typecheck
```
